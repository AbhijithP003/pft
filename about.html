About FEWD

A front-end web development course teaches students how to create the visual elements of websites and web applications. It covers HTML for structure, CSS for styling, and JavaScript for interactivity. Students learn to design responsive and user-friendly interfaces, optimize web performance, and use modern frameworks and libraries. Practical projects and hands-on coding exercises are typically included to reinforce concepts. Upon completion, students are equipped with the skills to build engaging and accessible web interfaces.

About The PFT™

Personal finance tracker is a simple yet useful tool which aims to provide college going students a platform to track their personal finances. What is included? A Home page. A TRACK NOW page which lets students input their personal finances along with date, and description. A separate page for displaying the entered data in the form of a graph. (ie. pie or line graph) And a final page which contains resources which would help them maintain healthy financial balance.
